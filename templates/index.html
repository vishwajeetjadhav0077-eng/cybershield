API Reference

All endpoints accept and return JSON. Replace YOUR_URL with your deployment URL.

GET /health

Check if the API is running.

"status": "ok",
"device": "cpu"
POST /predict

Analyze a single comment. Accepts JSON body {"text":"..."} or form-encoded text=....

"result":     1,          // 0 = safe, 1 = bullying
"category":   "gender",   // age | ethnicity | gender | religion | other_cyberbullying | not_cyberbullying
"confidence": 91.4,       // 0–100
"severity":   "high"      // none | low | medium | high
POST /predict/batch

Analyze up to 50 comments at once.

// Request
{ "texts": ["comment 1", "comment 2"] }

// Response — array in same order
[
  { "result": 0, "category": "not_cyberbullying", "confidence": 96.2, "severity": "none" },
  { "result": 1, "category": "ethnicity",         "confidence": 87.5, "severity": "high" }
]
GET /stats

Usage analytics — total checks, flagged rate, category breakdown.

Quick examples

# cURL
curl -X POST YOUR_URL/predict \
  -H "Content-Type: application/json" \
  -d '{"text":"your comment here"}'

# JavaScript (fetch)
const res = await fetch("YOUR_URL/predict", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ text: "your comment here" })
});
const data = await res.json();
console.log(data.result, data.category, data.confidence);

Chrome Extension Setup

1

Download the extension files

The extension/ folder contains 3 files: manifest.json, content.js, icons/. Keep them together in one folder.

2

Set your API URL

Open content.js and change line 1:

const API_URL = "https://your-deployed-url.com";

This tells the extension where to send comments for analysis.

3

Load in Chrome

  • Go to chrome://extensions
  • Turn on Developer Mode (top right)
  • Click Load unpacked
  • Select your extension/ folder
4

Supported sites

The extension works on:

  • YouTube — blurs toxic comments in comment threads
  • Instagram — scans post captions and visible comments
  • Twitter / X — scans tweet replies
  • Reddit — scans post comments

Hover over a blurred comment to preview, click to permanently reveal.

5

Deploy the API (free)

Recommended free platforms:

  • Render.com — connect your GitHub repo, set start command to gunicorn app:app
  • Railway.app — automatic deploys, free tier available
  • Hugging Face Spaces — great for ML models, free GPU available
